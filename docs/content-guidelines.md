# 于承惠双手剑内容编辑规范

更新日期：2026-07-20

状态：开发前基线

## 1. 适用范围

本规范适用于本站课程、动作名称、歌诀、人物生平、古籍原文、现代考释、教学插图、视频片段、图片说明、SEO 元数据和结构化数据。目标是让初学者读得懂，同时让研究者能够核对信息来自哪里、目前可靠到什么程度。

## 2. 核心编辑原则

1. 先说明已知事实，再说明解释和争议。
2. 初学者正文使用清楚的现代汉语，研究细节通过第二层内容展开。
3. 原始资料、现代考释和本站编辑说明不可混写成同一层。
4. 不确定的字词、动作名、年份或关系必须标明状态，不能靠语气掩盖不确定性。
5. 不把朝鲜势法二十四势和于承惠双手剑四十动作写成一套内容或逐项对应。
6. 教学插图以看懂动作结构为目的，不制作与内容无关的古风装饰。
7. 页面正文、图注、SEO 元数据和 JSON-LD 使用同一事实口径。

## 3. 内容状态

| 数据状态 | 页面标签 | 使用条件 | 发布规则 |
| --- | --- | --- | --- |
| `draft` | 草稿 | 内容结构或基本事实尚未完成 | 不生成正式页面，不进 sitemap |
| `needs-review` | 待校订 | 已按视频或资料整理，但尚未专业复核 | 可发布；标题附近必须显示状态 |
| `verified` | 已校订 | 已记录校订者、日期和依据 | 可发布并显示校订说明 |

与内容状态不同，来源或图片还需单独记录权利状态：

| 权利状态 | 含义 | 公开站处理 |
| --- | --- | --- |
| `cleared` | 已取得明确使用许可 | 可以发布 |
| `public-domain` | 已确认属于公版 | 可以发布，并注明底本 |
| `research-only` | 仅供内部研究参考 | 不得复制到生产资源 |
| `unknown` | 权利情况未知 | 不得发布图片或全文复制 |

## 4. 来源分级与引用

### 4.1 来源标签

| 标签 | 内容 | 示例 |
| --- | --- | --- |
| 原始影像 | 于承惠演练、讲解或正式出版影像 | 1996 年教学光盘相关影像 |
| 古籍原文 | 已核验的古籍影印或可靠整理 | 《武备志》卷八十六 |
| 出版资料 | 作者、书名、出版社和年份明确 | 《山东风物大全》 |
| 现代考释 | 后世作者对原典或技法的解释 | 源流观点、术语解释 |
| 编辑示意 | 本站据影像制作的图解或概述 | 剑路箭头、步法图 |

### 4.2 引用最低字段

- 书籍：作者或编者、书名、出版社、年份、卷页。
- 古籍：书名、卷次、底本机构或数据库、页码或稳定链接。
- 视频：名称、制作或出版信息、时间码、当前所用版本。
- 网页：作者或机构、标题、发布日期、URL、访问日期。
- 图片：作品或图版名称、底本、页码、提供者、权利状态。

无法补齐字段时写“待核验”，不得自行猜测。

### 4.3 正文引用写法

- 推荐：“视频字幕称，该套路共四段、40 个动作。”
- 推荐：“《武备志》卷八十六载‘近有好事者，得之朝鲜……’。”
- 推荐：“某现代文章认为……；该观点目前作为考释线索，尚未作为定论。”
- 避免：“众所周知”“据说”“公认”“中国最后的剑圣”等无法界定或核验的表述。

## 5. 动作与武术术语

- 动作名称先记录 `subtitleTitle`，即视频字幕原样转录。
- 专业校订后新增 `verifiedTitle`，不要覆盖原转录。
- 页面在待校订阶段显示字幕名并附“待校订”。
- 左、右、前、后以演练者自身方向为准；镜像模式必须明确说明方向已反转。
- 击、刺、格、洗属于技法分类时使用统一名称，不随意替换为近义词。
- 动作要领只描述视频和可靠资料能够支持的内容，不推演实战伤害效果。
- 常见问题使用练习性表达，例如“剑尖轨迹偏低”“转身与落脚不同步”。

## 6. 歌诀转录与校订

### 6.1 三层文本

1. 原字幕转录：尽量保留画面所见文字。
2. 校订文本：经更清晰母版、出版物或专业人员确认后的版本。
3. 校订说明：记录修改了什么、依据是什么、由谁在何时校订。

### 6.2 疑字格式

- 无法识别：`〔待辨〕`。
- 有候选字：正文保留暂拟字，注释写“‘某’待辨”。
- OCR 明显错误：仍在校订说明中记录原 OCR 结果，不把 OCR 当作原字幕。
- 每句必须保留可点击时间码；时间码属于定位信息，不等同于精确逐字字幕时间。

### 6.3 页面展示

- 标题附近显示“待校订”或“已校订”。
- 正文默认横排，古籍式竖排仅可作为附加视图。
- 视频不可用时，完整文字、时间码和来源仍可阅读。
- 播放中的当前句以实线、字重和语义状态突出，不只改变底色。

## 7. 人物生平

- 时间线每个节点至少关联一个来源。
- 年月日不确定时使用可支持的最小精度，例如“1996 年”，不补造月份。
- 区分出生、学习经历、创传过程、出版、影视活动和后世评价。
- 网络称号只能作为“后世称谓”并附来源，不能替代人物事实。
- 没有授权人物照片时使用授权视频截图或纯文字版式。
- 禁止使用 AI 生成的仿真历史照片冒充于承惠本人或历史场景。

## 8. 朝鲜势法专题

每个单势页面固定采用以下顺序：

1. 势名、序号和来源状态。
2. 图版及其来源/转载状态；第三方转载图不得标作公版或本站原创。
3. 古籍原文。
4. 现代标点。
5. 白话说明。
6. 术语注释。
7. 现代考释和争议。
8. 底本、卷次、页码、权利和更新时间。

必须明确说明：朝鲜势法二十四势与于承惠四段四十动作是两个内容系统。比较内容仅限有来源支持的术语、训练观念和技术类型。

## 9. 教学插图

### 9.1 类型标记

- “原始影像”：直接来自获授权视频的关键帧。
- “编辑示意”：从影像描摹、裁切、清理或加标注的图。
- “古籍图版”：来自已明确底本和许可的扫描。
- “注明来源转载”：按用户要求展示的第三方图版，必须逐图注明文章来源、未知底本和权利状态。
- “研究参考”：只存在于内部资料，不在公开页面展示。

### 9.2 图形语言

- 朱红实线箭头：剑尖或剑身运动方向。
- 钢蓝双线弧：肩、腰或身体旋转。
- 松绿足印与编号：落脚位置和先后。
- 虚线：回收路线或画面外延伸。
- 圆点：起点；箭头：终点。
- 永远保留左、右、前、后文字，不只使用颜色。

### 9.3 图片替代文本

alt 只描述图片传达的视觉信息，不堆关键词。

- 推荐：“击法示意，演练者双手持剑由右上向左下挥击，红色箭头标出剑路。”
- 推荐：“朝鲜势法举鼎势古谱图，人物双手举剑过头。”
- 避免：“于承惠双手剑武术教程高清学习击法双手剑教学图。”

纯装饰图使用空 alt；但本站原则上不生产无教学或史料用途的装饰插图。

## 10. 页面写作模板

### 10.1 通用内容页

1. 唯一 H1。
2. 40-100 个汉字的直接定义或结论。
3. 主要视频、图片或原文。
4. 分步骤说明。
5. 术语和相关内容。
6. 来源、状态、整理者和更新时间。
7. 待解决问题或修订记录。

### 10.2 动作教学页必须回答

- 动作目前叫什么，是否已校订？
- 属于第几段、第几组、第几个动作？
- 对应视频起止时间是什么？
- 起势、过渡和定势是什么？
- 使用了哪些基本技法？
- 剑路、步法和身体方向如何？
- 初学者常见问题是什么？
- 依据是原始影像、出版资料还是编辑示意？

## 11. SEO 与 GEO 文案

- 标题自然包含主体名称，不堆叠近义关键词。
- meta description 准确概括本页独有内容，不批量复制。
- 首段使用完整主语，例如“于承惠双手剑是……”。
- 数字、名称、年代旁边就近给出来源。
- 每个章节集中解决一个问题并使用稳定锚点。
- 普通 HTML 中保留关键文字，不能只放在视频、图片、Canvas 或弹窗中。
- JSON-LD 不添加页面看不见的作者、问答、评价或结论。
- `llms.txt` 只作为导航清单，不写排名承诺或特殊提示词。

## 12. 固定署名与占位文案

开发和预览阶段使用：

- 整理者：`本站整理`
- 视频说明：`来源与授权说明待补充`
- 动作状态：`依据视频字幕整理，待校订`
- 歌诀状态：`依据片尾字幕转录，疑字待校订`
- 古谱图片：`图版底本与使用许可待核验，暂不公开展示`

正式发布前，视频说明占位文案必须替换为具体来源与授权表述。

## 13. 修订记录

每次实质校订记录：

| 字段 | 要求 |
| --- | --- |
| 日期 | ISO 日期 `YYYY-MM-DD` |
| 内容对象 | 页面或条目稳定 ID |
| 修改前 | 原字幕、原稿或原说明 |
| 修改后 | 新文本 |
| 依据 | 视频时间码、书目页码或校订意见 |
| 校订者 | 姓名或公开署名 |
| 备注 | 仍有争议时说明范围 |

仅修复排版、链接或错别字且不改变事实含义时，可记录为技术更新；`updatedAt` 只在有实质变化时用于 sitemap `lastmod`。

## 14. 发布前内容检查

- 页面首段是否直接说明主题？
- 关键事实是否有来源？
- 待校订、待辨和待核验是否在页面可见？
- 原文、考释和编辑说明是否分层？
- 图片许可是否允许公开发布？
- alt 是否描述图像而非堆关键词？
- 动作左右方向和镜像说明是否一致？
- 页面正文与标题、摘要、JSON-LD 是否一致？
- 是否误建了朝鲜势法与现代套路的一一对应？
- 是否保留了修订依据和日期？
