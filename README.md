# 于承惠双手剑

面向初学者的双手剑公开科普与教学项目，以于承惠先生创传的双手剑为核心，结合教学影像、文字资料与古谱图版，整理基本技法、完整套路、歌诀和相关历史资料。

## 主要内容

- **双手剑二十法**：整理二十种基本剑法，建立双手剑术语与技法索引。
- **基本技法课程**：结合教学影像讲解击、刺、格、洗等主要剑法。
- **四十式套路**：按四段、十七个动作组整理完整套路，并提供四十式逐式学习页面。
- **双手剑歌诀**：收录教学影像片尾的单练行功歌诀及特点与演练要诀歌。
- **于承惠人物资料**：介绍于承惠先生的生平、武术实践与双手剑创传背景。
- **朝鲜势法专题**：整理《武备志·朝鲜势法》二十四势图文，呈现古谱原文、标点与完整图版索引。

## 内容定位

项目以初学者阅读和观看为第一层，以人物、古籍和研究资料为延伸内容。现代四十式与《武备志·朝鲜势法》分别编排，不将两套目录强行建立动作对应关系。

教学页面不要求登录，也不记录学习进度。内容用于公开科普、资料查阅和辅助学习，器械练习应在安全空间内进行，并由具备经验的教师现场指导。

## 资料基础

项目内容主要依据于承惠先生双手剑教学影像、影像中的剑招与歌诀文本、双手剑二十法资料，以及《武备志·朝鲜势法》相关公开图文资料整理。
